template
========
  作業効率化のための自分用テンプレート。
  随時アップデートしております。

##Usage
###1. Install Sass & Compass
以下を実行して、*Sass*と*Compass*をローカルにインストールします。
`$ bundle install`

###2. npm install
以下を実行して、`package.json`の内容をインストール。
`$ npm install`

##Grunt command
$ grunt dev   開発用
$ grunt style   スタイルガイド生成用
$ grunt sprite  スプライト画像生成用
$ grunt ass     アセンブル（テンプレート）構築用
$ grunt html    HTMLリント
$ grunt css     CSSリント
