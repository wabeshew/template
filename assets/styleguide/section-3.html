nav

Nav styles can be applied to anything with the .nav class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

3.1 Heading

ナビゲーションのモジュールです。

  • top
  • about
  • works
  • acsess
  • contact
<div class="nav1_1">
    <nav class="gnav {$modifiers}">
        <ul>
            <li><a href="#" class="gnav-menu">top</a></li>
            <li><a href="#" class="gnav-menu">about</a></li>
            <li><a href="#" class="gnav-menu">works</a></li>
            <li><a href="#" class="gnav-menu">acsess</a></li>
            <li><a href="#" class="gnav-menu">contact</a></li>
        </ul>
    </nav>
</div>
