Buttons

Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

見出しのモジュールです。

検討候補に登録している施設はありません
興味をもった施設があったら、検討候補に追加してみましょう。検討候補には最大20件まで登録することができます。
施設を検討候補に登録すると、複数の施設を比較検討したり、まとめて資料請求や見学予約をしたりできます。

  • 一覧ページや施設詳細ページで「興味あり」ボタンを押すと、「検討候補一覧」に登録できます。

  • 比較表を利用すれば、費用や医療受け入れ体制など、気になる箇所を比較できます。

  • ご希望の施設には、まとめて資料請求したり、見学予約したりできます。
